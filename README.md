#小專題題目:酷酷蛇
#程式介紹:
###酷酷蛇:主要基於貪吃蛇改造而來，把原本的撞牆與撞自己就死改成了五滴血扣完才結束，且撞到自己會扣一滴血及蛇生會撞到的地方斷開，遊戲中有小機率會出現金蘋果回血，移動會隨著遊戲進行加速。
